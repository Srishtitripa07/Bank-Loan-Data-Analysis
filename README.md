This project analyzes bank loan data to uncover insights related to loan applications, funded amounts, repayment performance, customer risk, and loan distribution trends.

Using SQL for data extraction and transformation and Tableau for visualization, an interactive dashboard was developed to monitor key loan KPIs and business metrics.

The dashboard is divided into two main sections:

1. Summary Report

2. Overview Report

## Tools & Technologies Used

SQL – Data cleaning, transformation, aggregation, KPI calculations

Tableau – Interactive dashboard development and visualization

Excel/CSV Dataset – Source data

## Dashboard 1: Summary Report

The Summary Dashboard provides high-level performance indicators of the bank loan portfolio.

🔹 Key KPIs Tracked:

Total Loan Applications (38.6K)

Total Funded Amount ($435.8M)

Total Amount Received ($473.1M)

Average Interest Rate (12.0%)

Average Debt-to-Income Ratio (13.3%)

Month-to-Date (MTD) and Month-over-Month (MoM) growth metrics

🔹 Good vs Bad Loan Analysis:

 1.Good Loans: 86.2%

2. Bad Loans: 13.8%

Additional breakdown includes:

Funded Amount by Loan Status

Amount Received by Loan Status

Interest Rate & DTI comparison

Charged Off vs Fully Paid vs Current loans

This dashboard helps in evaluating:

Portfolio health

Default risk exposure

Repayment efficiency

Credit performance trends

## Dashboard 2: Overview Report

The Overview Dashboard provides deeper insights into loan distribution and customer segmentation.

 Monthly Trend Analysis

Total Amount Received by Month

Growth trend across the year

 Geographic Analysis

Total Amount Received by State (Map visualization)

 Loan Term Analysis

36 months vs 60 months loan distribution
 Customer Segmentation

Loan Purpose (Debt consolidation, Credit card, Home improvement, etc.)

Employee Length

Home Ownership (Mortgage, Rent, Own)

This dashboard helps stakeholders understand:

Customer behavior patterns

High-performing states

Loan term preference

Risk segmentation by employment and ownership
